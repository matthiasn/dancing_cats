# Motion-review render imagery

Before/after frame-grids and onion-skins from the 5-lens character-motion review
panel are **not committed to this repo** — they live in the separate
[`dancing_cats-docs`](https://github.com/matthiasn/dancing_cats-docs) repo
(PNG, on `main`) to keep this repo lean.

Reference them from PR bodies via raw URL:

```
https://raw.githubusercontent.com/matthiasn/dancing_cats-docs/main/images/reviews/<name>.png
```

Because they live on `dancing_cats-docs`' `main` (not a feature branch here),
the URLs are permanent and survive branch deletion — so PR review images no
longer depend on keeping merged feature branches around.
